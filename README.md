# Gamba Simple Plinko

Plain HTML, CSS, and JavaScript Plinko game.

No npm.
No Vite.
No React.
No GitHub Actions.

## Files

Upload these files to your GitHub repository root:

```txt
index.html
styles.css
app.js
README.md
```

## GitHub Pages Setup

1. Open the GitHub repository.
2. Upload the files above.
3. Go to `Settings`.
4. Go to `Pages`.
5. Set `Source` to `Deploy from a branch`.
6. Choose branch `main`.
7. Choose folder `/ (root)`.
8. Save.

Your site will be available at:

```txt
https://YOUR_USERNAME.github.io/YOUR_REPO_NAME/
```

If the repo is named `Gamba`, it will be:

```txt
https://YOUR_USERNAME.github.io/Gamba/
```

## Payout Model

Each risk uses a pure percentage table that totals `100.00%`.

- Green: `60.00% RTP`, highest multiplier chance `1.00%` per edge slot.
- Yellow: `40.00% RTP`, highest multiplier chance `0.10%` per edge slot.
- Red: `20.00% RTP`, highest multiplier chance `0.01%` per edge slot.

The visible landing slot and the payout slot are the same slot.
