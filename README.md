# Gamba Plinko

This version is intentionally simple: one self-contained `index.html` file.

Upload `index.html` from this `Gamba` folder to your GitHub repo root.

GitHub Pages settings:

- Source: Deploy from a branch
- Branch: main
- Folder: / root

Do not use GitHub Actions for this simple version.
Do not upload npm/node/Vite files for this version.
